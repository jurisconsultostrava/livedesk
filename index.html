Best bid je cena, za kterou může klient prodávat. Best offer je cena, za kterou může klient nakupovat.

Live ceny se načítají přes backend endpoint /.netlify/functions/metals-prices. Pokud chybí API klíč, systém spadne do demo režimu.

Admin Console umožňuje správu uživatelů, objednávek, marží, audit logu a systémových kontrol. V této verzi jde o frontendový prototyp s localStorage.
