# Order Board Glossary CZ/EN

| English | Czech UI translation | Poznámka |
|---|---|---|
| Market | Trh | konkrétní kombinace kov + lokace + měna |
| Currency | Měna | měna vypořádání / kotace |
| Best bid | Nejlepší poptávka | nejlepší cena, za kterou je někdo připraven nakoupit; klient proti ní prodává |
| Best offer | Nejlepší nabídka | nejlepší cena, za kterou je někdo připraven prodat; klient proti ní nakupuje |
| Bid quantity | Množství v poptávce | objem kovu dostupný na straně poptávky |
| Offer quantity | Množství v nabídce | objem kovu dostupný na straně nabídky |
| Indicative liquidity | Orientační likvidita | orientační disponibilní objem v daném trhu |
| Market depth | Hloubka trhu | více cenových úrovní bid/offer |
| Spread | Spread | rozdíl mezi nejlepší nabídkou a nejlepší poptávkou |
| Spot | Spot | základní spotová cena kovu |
| Fee | Poplatek | obchodní poplatek nebo commission reserve |
| Settlement | Vypořádání | interní evidence, alokovaný kov, fyzické dodání |
| Order | Pokyn / objednávka | podle kontextu: obchodní pokyn nebo zákaznická objednávka |
| Draft | Draft / rozpracováno | uložený, nepotvrzený pokyn |
| Pending | Čeká na schválení | pokyn čeká na kontrolu |
| Approved | Schváleno | schváleno dealerem/adminem |
| Matched | Spárováno | spárováno / vypořádáno |
| Rejected | Odmítnuto | neprovedeno |
| Cancelled | Zrušeno | zrušeno klientem/adminem |
